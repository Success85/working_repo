About Rocel

A student-built finance tracker for real student life.

◎

Purpose

Rocel helps students take control of their money — track income from jobs and scholarships, log daily expenses, visualize trends, and export their data for further analysis. Built with accessibility and mobile-first design in mind.

◉

Features

  • Add income & expense transactions
  • Regex-powered live search with highlights
  • Sort by date, amount, or description
  • Budget cap with ARIA live alerts
  • 7-day spending trend chart
  • JSON import/export with validation
  • Full keyboard navigation support
  • localStorage persistence
  • Multi-currency display (Settings)
⚙

Tech Stack

  • Vanilla HTML5, CSS3, JavaScript (ES Modules)
  • No frameworks or external libraries
  • Mobile-first responsive design
  • WCAG 2.1 AA accessibility target
  • localStorage for data persistence

Regex Catalog

Rule Pattern Example
Description (no leading/trailing spaces) /^\S(?:.*\S)?$/ "Coffee with friends" ✓
Amount (currency) /^(0|[1-9]\d*)(\.\d{1,2})?$/ "12.50" ✓, "0.5" ✓
Date (YYYY-MM-DD) /^\d{4}-(0[1-9]|1[0-2])-(0[1-9]|[12]\d|3[01])$/ "2025-09-29" ✓
Category (letters/spaces/hyphens) /^[A-Za-z]+(?:[ -][A-Za-z]+)*$/ "Part-time" ✓
Duplicate word (advanced — back-reference) /\b(\w+)\s+\1\b/i "the the" detected ✓
Cents present (search) /\.\d{2}\b/ Finds "12.50", "8.75"
Beverage keyword (search) /(coffee|tea)/i Finds "Coffee with friends"
